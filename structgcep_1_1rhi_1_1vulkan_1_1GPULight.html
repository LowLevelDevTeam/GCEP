Detailed Description

Tightly-packed GPU light entry. Laid out as std430 so the Slang StructuredBuffer can address it with sizeof == 64 bytes.

Both PointLight and SpotLight are packed into a single union-like structure. Fields unused by a given light type are zeroed on upload.
