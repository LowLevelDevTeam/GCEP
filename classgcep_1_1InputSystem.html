Detailed Description

Polls registered GLFW key states and dispatches callbacks each frame.

InputSystem provides a lightweight, callback-driven abstraction over raw GLFW key polling. Callers register (key, callback) pairs via addTrackedKey(). Each call to update() polls every registered key; when a key is held down (GLFW_PRESS), its associated callback is invoked immediately on the calling thread.

Typical usage
InputSystem input(window);
input.addTrackedKey(GLFW_KEY_W, [&]{ camera.moveForward(); });
input.addTrackedKey(GLFW_KEY_ESCAPE, [&]{ running = false; });
while (running)
{
glfwPollEvents();
input.update(); // fires callbacks for all pressed keys
}

void gcep::InputSystem::update ( )

Polls all registered keys and fires their callbacks if pressed.

For each entry in m_trackedKeys, calls glfwGetKey() and invokes the stored callback when the result is GLFW_PRESS. Must be called once per frame after glfwPollEvents().
