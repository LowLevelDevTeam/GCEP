Detailed Description

Singleton that owns the GLFW window for the entire editor process.

Window wraps a GLFWwindow* and exposes the minimal interface needed by the rest of the editor: initialisation, the native handle, close-request polling, and destruction.

The class is a Meyer's singleton: the unique instance is created on the first call to getInstance() and lives until program exit.

Typical usage
Window& win = Window::getInstance();
win.initWindow();
while (!win.shouldClose())
{
glfwPollEvents();
// render frame ...
}
win.destroy();

void gcep::Window::initWindow ( )

Creates and displays the GLFW window.

Initialises GLFW, sets Vulkan-specific window hints (no OpenGL context), creates the window with a hard-coded default resolution, sets the window icon, and centres it on the primary monitor via centerWindow().

Note
Must be called exactly once before any other method.
