\subsubsection*{template$<$typename T$>$\newline
class gcep::\+\+Mat3$<$ T $>$}
Represents a 3x3 matrix for linear algebra operations. 

This class supports matrix creation, multiplication, inversion, and transformations such as rotation and scaling. The matrix is stored in {\bfseries{row-\/major order}}.
