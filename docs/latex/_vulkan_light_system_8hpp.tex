\doxysubsection{Detailed Description}
CPU-\/side light data types, GPU-\/mapped buffer structs, and the {\ttfamily Light\+System} class that owns all Vulkan resources for point and spot lights. 

\begin{DoxyParagraph}{Architecture overview}
The light system is a self-\/contained subsystem that plugs directly into {\ttfamily Vulkan\+RHI}. It owns:\+
\begin{DoxyItemize}
\item A persistently-\/mapped, host-\/visible/\+coherent SSBO containing packed {\ttfamily GPULight} entries (one per live light, up to {\ttfamily MAX\+\_\+\+LIGHTS}).
\item A host-\/visible/\+coherent metadata UBO ({\ttfamily Light\+Meta\+UBO}) that carries the live light count so the lighting compute shader can early-\/exit.
\item A compute pipeline ({\ttfamily Compute\+\_\+\+Lighting.\+spv}, entry {\ttfamily comp\+Main}) that reads the G-\/buffer-\/like offscreen color + depth produced by the scene pass, the light SSBO, and writes a shaded output into an RGBA16F storage image. The result is fed into the TAA resolve stage as the new "{}current color"{}.
\item All descriptor set layouts, pools, and sets needed for the above.
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Descriptor set layout (set 3 -\/ lighting compute)}

\begin{DoxyItemize}
\item Binding 0 {\ttfamily e\+Storage\+Buffer} -\/ {\ttfamily GPULight} SSBO (read-\/only)
\item Binding 1 {\ttfamily e\+Uniform\+Buffer} -\/ {\ttfamily Light\+Meta\+UBO} (read-\/only)
\item Binding 2 {\ttfamily e\+Combined\+Image\+Sampler} -\/ offscreen color (read)
\item Binding 3 {\ttfamily e\+Combined\+Image\+Sampler} -\/ offscreen depth (read)
\item Binding 4 {\ttfamily e\+Storage\+Image} -\/ lit output (write)
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Push constants ({\ttfamily Lighting\+Push\+Constants}, compute stage)}

\begin{DoxyItemize}
\item {\ttfamily inv\+View\+Proj} -\/ inverse view-\/projection for depth-\/buffer ray reconstruction
\item {\ttfamily camera\+Pos} -\/ world-\/space eye position for specular computation
\item {\ttfamily screen\+Size} -\/ pixel dimensions of the offscreen render target
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Integration in {\ttfamily Vulkan\+RHI::\+record\+Offscreen\+Command\+Buffer}()}
Insert {\ttfamily record\+Lighting\+Pass()} {\itshape after} {\ttfamily end\+Rendering()} and {\itshape before} {\ttfamily record\+TAAPass()}. The lighting pass reads the scene color and depth as sampled images and writes the lit result into {\ttfamily m\+\_\+lit\+Image} (storage), then transitions it to {\ttfamily e\+Shader\+Read\+Only\+Optimal} for TAA.
\end{DoxyParagraph}
\begin{DoxyParagraph}{Thread safety}
All public methods must be called from the render thread. No internal synchronisation is provided. 
\end{DoxyParagraph}
