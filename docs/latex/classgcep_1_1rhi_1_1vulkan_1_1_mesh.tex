\doxysubsection{Detailed Description}
CPU-\/side geometry, transform, and texture data for a single 3D mesh entity. 

Handles the full pipeline from OBJ loading to GPU upload readiness:\+ vertex deduplication, AABB computation, and CPU-\/side buffer storage. The actual device-\/local buffers live in {\ttfamily \doxylink{classgcep_1_1rhi_1_1vulkan_1_1_vulkan_r_h_i}{Vulkan\+RHI}\textquotesingle{}s} global vertex and index buffers; this class only holds staging data and metadata until {\ttfamily \doxylink{classgcep_1_1rhi_1_1vulkan_1_1_mesh_a61cd2da28f81178a18cc12020b391985}{clear\+Vertices()}} /\+ {\ttfamily \doxylink{classgcep_1_1rhi_1_1vulkan_1_1_mesh_ab7bf8407b11cdb17ca936d4d0278f8d6}{clear\+Indices()}} are called after the GPU upload.

Instances are stored by value inside {\ttfamily Vulkan\+RHI::\+meshes}. Each instance owns exactly one {\ttfamily \doxylink{classgcep_1_1rhi_1_1vulkan_1_1_vulkan_texture}{Vulkan\+Texture}} which may internally share GPU resources with other instances via {\ttfamily \doxylink{classgcep_1_1rhi_1_1vulkan_1_1_vulkan_texture}{Vulkan\+Texture}\textquotesingle{}s} {\ttfamily shared\+\_\+ptr-\/based} cache.

\begin{DoxyNote}{Note}
Holds a non-\/owning pointer to {\ttfamily \doxylink{classgcep_1_1rhi_1_1vulkan_1_1_vulkan_r_h_i}{Vulkan\+RHI}} — the \doxylink{classgcep_1_1_r_h_i}{RHI} must outlive all meshes it owns. 
\end{DoxyNote}

Loads an OBJ file and optionally a texture from disk. 

Parses the OBJ via {\ttfamily Obj\+Loader}, builds the vertex array, computes the object-\/space AABB, and loads the texture via {\ttfamily \doxylink{classgcep_1_1rhi_1_1vulkan_1_1_vulkan_texture_a28da1fadcbd3b52c50e2d2aae97eb4dd}{Vulkan\+Texture::\+load\+Texture()}} if {\ttfamily texture\+Filepath} is non-\/empty. CPU attribute data is freed immediately after the vertex array is built. {\ttfamily m\+\_\+vertices} and {\ttfamily m\+\_\+indices} remain populated until {\ttfamily Vulkan\+RHI::\+upload\+Single\+Mesh()} calls {\ttfamily \doxylink{classgcep_1_1rhi_1_1vulkan_1_1_mesh_a61cd2da28f81178a18cc12020b391985}{clear\+Vertices()}} and {\ttfamily \doxylink{classgcep_1_1rhi_1_1vulkan_1_1_mesh_ab7bf8407b11cdb17ca936d4d0278f8d6}{clear\+Indices()}}.
