class gcep::\+\+ECS::\+\+View$<$ Args $>$}
Query system used to iterate over entities possessing a specific set of components. 

The \doxylink{classgcep_1_1_e_c_s_1_1_view}{View} is the primary tool for entity filtering. It optimizes iteration by identifying the smallest (most restrictive) pool to minimize the number of checks performed.
