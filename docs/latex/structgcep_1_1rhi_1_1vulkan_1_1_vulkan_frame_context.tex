\doxysubsection{Detailed Description}
Owns all per-\/frame-\/in-\/flight Vulkan objects for a single frame slot. 

With {\ttfamily MAX\+\_\+\+FRAMES\+\_\+\+IN\+\_\+\+FLIGHT} = 2 the CPU can record frame N+1 while the GPU is still executing frame N, without ever stalling on the second frame.

\begin{DoxyParagraph}{Member roles}

\begin{DoxyItemize}
\item {\ttfamily command\+Pool} -\/ Created with {\ttfamily e\+Transient}; reset wholesale each frame via {\ttfamily vk\+Reset\+Command\+Pool}, which is faster than resetting individual buffers.
\item {\ttfamily command\+Buffer} -\/ Primary buffer allocated from the pool; re-\/recorded every frame.
\item {\ttfamily command\+List} -\/ Typed wrapper around {\ttfamily command\+Buffer}.
\item {\ttfamily in\+Flight\+Fence} -\/ Pre-\/signaled at construction so frame 0 does not stall before the GPU has done any work. Waited on and reset in {\ttfamily \doxylink{structgcep_1_1rhi_1_1vulkan_1_1_vulkan_frame_context_ac0cc8d881276e86a3735367f24bac8e2}{wait\+And\+Reset()}} before each recording.
\item {\ttfamily present\+Complete\+Semaphore} -\/ Signaled by {\ttfamily vk\+Acquire\+Next\+Image\+KHR} when the swapchain image is safe to write. The GPU waits at {\ttfamily e\+Color\+Attachment\+Output} before writing color.
\end{DoxyItemize}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Destruction order}
Member declaration order is also the reverse-\/destruction order enforced by C++. {\ttfamily command\+List} -\/\texorpdfstring{$>$}{>} {\ttfamily command\+Buffer} -\/\texorpdfstring{$>$}{>} {\ttfamily command\+Pool} -\/\texorpdfstring{$>$}{>} {\ttfamily in\+Flight\+Fence} -\/\texorpdfstring{$>$}{>} {\ttfamily present\+Complete\+Semaphore}.
\end{DoxyParagraph}
\begin{DoxyParagraph}{Thread safety}
Not thread-\/safe. Owned and used exclusively by the render thread. 
\end{DoxyParagraph}

Constructs and initialises all per-\/frame Vulkan objects. 

Allocates a transient command pool for {\ttfamily queue\+Family}, allocates a single primary command buffer from it, creates a pre-\/signaled in-\/flight fence, and creates the acquire semaphore. The {\ttfamily \doxylink{classgcep_1_1rhi_1_1vulkan_1_1_vulkan_command_list}{Vulkan\+Command\+List}} is constructed last, wrapping the allocated command buffer.

Blocks the CPU until the GPU finishes all work submitted with this frame slot\textquotesingle{}s fence, then resets the fence and command pool. 

This is the "{}begin of frame"{} CPU stall. Calling {\ttfamily vk\+Reset\+Command\+Pool} in one shot is faster than resetting individual command buffers because the driver can reclaim all allocator memory at once.
