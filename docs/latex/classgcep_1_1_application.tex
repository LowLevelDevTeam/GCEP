\doxysubsection{Detailed Description}
Top-\/level application class that owns and orchestrates all editor subsystems. 

{\ttfamily \doxylink{classgcep_1_1_application}{Application}} is the root object of the editor process. It is responsible for:\+
\begin{DoxyItemize}
\item Initialising the window, \doxylink{classgcep_1_1_r_h_i}{RHI}, input system, camera, Im\+Gui, and UI manager.
\item Running the main loop and computing per-\/frame delta time.
\item Gracefully shutting down every subsystem in reverse-\/initialisation order.
\end{DoxyItemize}
