\doxysubsection{Detailed Description}
First-\/person editor camera that maps keyboard/\+mouse events to view and projection matrices consumed by the \doxylink{classgcep_1_1_r_h_i}{RHI}. 

{\ttfamily \doxylink{classgcep_1_1_camera}{Camera}} polls an {\ttfamily \doxylink{classgcep_1_1_input_system}{Input\+System}} each frame to translate WASD /\+ QE keyboard input into positional movement and drag-\/based mouse input into yaw/\+pitch rotation. The resulting matrices are packed into a {\ttfamily \doxylink{structgcep_1_1rhi_1_1vulkan_1_1_uniform_buffer_object}{rhi::\+vulkan::\+\+Uniform\+Buffer\+Object}} and returned by {\ttfamily \doxylink{classgcep_1_1_camera_a37a93059bf2497140ca65a86eea3371f}{update()}}.

\begin{DoxyParagraph}{Coordinate system}

\begin{DoxyItemize}
\item The world-\/up axis is {\ttfamily }(0, 0, 1) (Z-\/up).
\item The camera\textquotesingle{}s local up is {\ttfamily }(0, 1, 0).
\item Yaw is measured from the negative-\/X axis, pitch from the XY plane.
\end{DoxyItemize}
\end{DoxyParagraph}

Advances the camera simulation by one frame and returns the updated UBO. 

Processes movement keys (WASD /\+ QE) scaled by {\ttfamily cam\+Speed}, updates yaw/\+pitch from right-\/mouse-\/button drag, then recomputes {\ttfamily front}, {\ttfamily right}, and {\ttfamily velocity}. Writes the new view/\+projection matrices and the camera position into {\ttfamily ubo} before returning it.
