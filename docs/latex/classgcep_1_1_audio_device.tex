\doxysubsection{Detailed Description}
Audio device wrapper around miniaudio. 

\doxylink{classgcep_1_1_audio_device}{Audio\+Device} is responsible for:\+
\begin{DoxyItemize}
\item Initializing and shutting down the audio hardware
\item Managing the miniaudio playback device
\item Forwarding audio callbacks to the \doxylink{classgcep_1_1_audio_mixer}{Audio\+Mixer}
\end{DoxyItemize}

This class does not perform any mixing by itself. All audio mixing logic is delegated to \doxylink{classgcep_1_1_audio_mixer}{Audio\+Mixer}.

There should generally be only one \doxylink{classgcep_1_1_audio_device}{Audio\+Device} instance for the entire engine.
