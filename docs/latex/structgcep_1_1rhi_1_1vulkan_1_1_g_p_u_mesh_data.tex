\doxysubsection{Detailed Description}
Per-\/mesh data stored in the GPU-\/side SSBO (set 0, binding 0). 

One entry per mesh, indexed in the vertex shader via {\ttfamily SV\+\_\+\+Start\+Instance\+Location} which carries the {\ttfamily first\+Instance} value set in each {\ttfamily Draw\+Indexed\+Indirect\+Command}.

Must be kept byte-\/for-\/byte identical to the {\ttfamily Object\+Data} /\+ {\ttfamily \doxylink{structgcep_1_1rhi_1_1vulkan_1_1_g_p_u_mesh_data}{GPUMesh\+Data}} struct in both {\ttfamily Base\+\_\+\+Shader.\+slang} and {\ttfamily Cull.\+slang}.

\begin{DoxyNote}{Note}
The two {\ttfamily float} padding fields ({\ttfamily \+\_\+pad0}, {\ttfamily \+\_\+pad1}) are required to satisfy the std430 alignment rule for {\ttfamily vec3} members inside a storage buffer. 
\end{DoxyNote}
