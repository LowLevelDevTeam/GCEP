\doxysubsection{Detailed Description}
Central Vulkan device manager. 

Owns the Vulkan instance, debug messenger, surface, physical device, logical device, queue, swapchain, and all per-\/frame-\/in-\/flight contexts. Coordinates the full frame lifecycle (acquire -\/\texorpdfstring{$>$}{>} record -\/\texorpdfstring{$>$}{>} submit -\/\texorpdfstring{$>$}{>} present).

\begin{DoxyParagraph}{Init sequence}
{\ttfamily \doxylink{classgcep_1_1rhi_1_1vulkan_1_1_vulkan_device_acc11ffb5acf9b126c1cb0e49dcc7a9ce}{create\+Vulkan\+Device()}} must be the first call after construction. It runs all eight init stages in mandatory order:\+ {\ttfamily init\+Instance} -\/\texorpdfstring{$>$}{>} {\ttfamily init\+Debug\+Messenger} -\/\texorpdfstring{$>$}{>} {\ttfamily init\+Surface} -\/\texorpdfstring{$>$}{>} {\ttfamily select\+Physical\+Device} -\/\texorpdfstring{$>$}{>} {\ttfamily init\+Logical\+Device} -\/\texorpdfstring{$>$}{>} {\ttfamily init\+Swapchain} -\/\texorpdfstring{$>$}{>} {\ttfamily init\+Frame\+Contexts}.
\end{DoxyParagraph}

Begins a new frame:\+ waits on the in-\/flight fence, acquires a swapchain image, and opens the frame\textquotesingle{}s command buffer for recording. 

Selects the next frame slot (round-\/robin over {\ttfamily MAX\+\_\+\+FRAMES\+\_\+\+IN\+\_\+\+FLIGHT}), calls {\ttfamily wait\+And\+Reset()} on its {\ttfamily \doxylink{structgcep_1_1rhi_1_1vulkan_1_1_vulkan_frame_context}{Vulkan\+Frame\+Context}} to stall the CPU until the GPU is done with that slot, then signals {\ttfamily present\+Complete\+Semaphore} via {\ttfamily vk\+Acquire\+Next\+Image\+KHR} and calls {\ttfamily begin\+Recording()} on the command list.

\begin{DoxyReturn}{Returns}
{\ttfamily true} if recording can proceed; {\ttfamily false} if the swapchain is out-\/of-\/date ({\ttfamily e\+Error\+Out\+Of\+Date\+KHR}), in which case the caller must skip the rest of the frame. 
\end{DoxyReturn}
