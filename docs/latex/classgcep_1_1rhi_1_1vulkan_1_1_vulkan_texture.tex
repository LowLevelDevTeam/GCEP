\doxysubsection{Detailed Description}
Manages the lifetime and GPU resources of a 2D Vulkan texture. 

Handles the full pipeline from loading an image file off disk to registering it in the global bindless descriptor array:\+ staging upload, optional mip-\/chain generation, image view creation, sampler configuration, and bindless slot registration.

Typical usage:\+ 
\begin{DoxyCode}{0}
\DoxyCodeLine{VulkanTexture\ tex;}
\DoxyCodeLine{tex.loadTexture(rhi,\ \textcolor{stringliteral}{"{}assets/albedo.png"{}},\ \textcolor{keyword}{true});}
\DoxyCodeLine{uint32\_t\ slot\ =\ tex.getBindlessIndex();\ \textcolor{comment}{//\ pass\ to\ GPUMeshData::textureIndex}}

\end{DoxyCode}


\begin{DoxyNote}{Note}
The texture holds a non-\/owning pointer to the {\ttfamily \doxylink{classgcep_1_1rhi_1_1vulkan_1_1_vulkan_r_h_i}{Vulkan\+RHI}} instance that created it; the \doxylink{classgcep_1_1_r_h_i}{RHI} must therefore outlive any {\ttfamily \doxylink{classgcep_1_1rhi_1_1vulkan_1_1_vulkan_texture}{Vulkan\+Texture}} it owns. 
\end{DoxyNote}

void gcep::\+rhi::\+vulkan::\+\+Vulkan\+Texture::\+load\+Texture (\begin{DoxyParamCaption}\item[{\doxymbox{\hyperlink{classgcep_1_1rhi_1_1vulkan_1_1_vulkan_r_h_i}{Vulkan\+RHI}} \texorpdfstring{$\ast$}{*}}]{instance}{, }\item[{const std::\+filesystem::\+path \&}]{filepath}{, }\item[{bool}]{mipmaps}{ = {\ttfamily true}}\end{DoxyParamCaption})}



Loads an image from disk and registers it in the global bindless array. 

Internally allocates a host-\/visible staging buffer, copies pixel data into it, transitions the image layout to {\ttfamily e\+Transfer\+Dst\+Optimal}, submits a buffer-\/to-\/image copy, and — when {\ttfamily mipmaps} is {\ttfamily true} — generates a full mip chain via {\ttfamily generate\+Mipmaps()}. The final layout is {\ttfamily e\+Shader\+Read\+Only\+Optimal}. Finally calls {\ttfamily register\+Bindless()} to write the image view + sampler into the bindless descriptor set at an atomically allocated slot.

Dynamically adjusts the minimum LOD bias of the sampler. 

Waits for the device to be idle, recreates the {\ttfamily Vk\+Sampler} with {\ttfamily lod\+Level} as the {\ttfamily min\+Lod} clamp, and re-\/registers the texture in the bindless set at its existing slot. Useful for visualising individual mip levels or implementing LOD-\/fade effects at runtime.

Only has an effect if the texture was loaded with {\ttfamily mipmaps} = {\ttfamily true} and {\ttfamily \doxylink{classgcep_1_1rhi_1_1vulkan_1_1_vulkan_texture_a28da1fadcbd3b52c50e2d2aae97eb4dd}{load\+Texture()}} has completed successfully.
