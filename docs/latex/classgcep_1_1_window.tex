\doxysubsection{Detailed Description}
Singleton that owns the GLFW window for the entire editor process. 

{\ttfamily \doxylink{classgcep_1_1_window}{Window}} wraps a {\ttfamily GLFWwindow\texorpdfstring{$\ast$}{*}} and exposes the minimal interface needed by the rest of the editor:\+ initialisation, the native handle, close-\/request polling, and destruction.

The class is a Meyer\textquotesingle{}s singleton:\+ the unique instance is created on the first call to {\ttfamily \doxylink{classgcep_1_1_window_ad720021e4ddfae574fdb4278620eb1c2}{get\+Instance()}} and lives until program exit.

Creates and displays the GLFW window. 

Initialises GLFW, sets Vulkan-\/specific window hints (no Open\+GL context), creates the window with a hard-\/coded default resolution, sets the window icon, and centres it on the primary monitor via {\ttfamily center\+Window()}.
