\doxysubsection{Detailed Description}
Tightly-\/packed GPU light entry. Laid out as std430 so the Slang {\ttfamily Structured\+Buffer} can address it with {\ttfamily sizeof} == 64 bytes. 

Both {\ttfamily \doxylink{structgcep_1_1rhi_1_1vulkan_1_1_point_light}{Point\+Light}} and {\ttfamily \doxylink{structgcep_1_1rhi_1_1vulkan_1_1_spot_light}{Spot\+Light}} are packed into a single union-\/like structure. Fields unused by a given light type are zeroed on upload.
