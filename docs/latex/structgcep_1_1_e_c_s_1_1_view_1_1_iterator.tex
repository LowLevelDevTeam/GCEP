Custom iterator for traversing entities validated by the \doxylink{classgcep_1_1_e_c_s_1_1_view}{View}. 

This iterator traverses the dense array of the smallest pool contiguously and automatically skips entities that fail the \doxylink{classgcep_1_1_e_c_s_1_1_view_a800c8f7f59977d0400c885aeca91ad04}{match()} test.
