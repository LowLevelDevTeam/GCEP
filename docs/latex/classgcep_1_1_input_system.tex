\doxysubsection{Detailed Description}
Polls registered GLFW key states and dispatches callbacks each frame. 

{\ttfamily \doxylink{classgcep_1_1_input_system}{Input\+System}} provides a lightweight, callback-\/driven abstraction over raw GLFW key polling. Callers register {\ttfamily }(key, callback) pairs via {\ttfamily \doxylink{classgcep_1_1_input_system_ac5bc36aed585b131ffb6e6e54503b469}{add\+Tracked\+Key()}}. Each call to {\ttfamily \doxylink{classgcep_1_1_input_system_a2fa29188998ebc92cfa01eb9f68e2c1c}{update()}} polls every registered key; when a key is held down (GLFW\+\_\+\+PRESS), its associated callback is invoked immediately on the calling thread.

\begin{DoxyNote}{Note}
{\ttfamily \doxylink{classgcep_1_1_input_system_a2fa29188998ebc92cfa01eb9f68e2c1c}{update()}} must be called after {\ttfamily glfw\+Poll\+Events()} so that GLFW\textquotesingle{}s internal key state is up-\/to-\/date. 

Only one callback per key is supported; registering the same key twice overwrites the previous entry.

Polls all registered keys and fires their callbacks if pressed.
