\doxysubsection{Detailed Description}
Singleton responsible for project lifecycle management. 

{\ttfamily \doxylink{classpl_1_1_project_loader}{Project\+Loader}} is the single point of truth for the active project. It:\+
\begin{DoxyItemize}
\item Presents an Im\+Gui project-\/selection /\+ creation UI at startup.
\item Reads and writes the {\ttfamily }.gcepproj JSON file via Rapid\+JSON.
\item Exposes runtime settings through {\ttfamily \doxylink{classpl_1_1_project_loader_adb4c96f0ed364e54bd33654dc82c9045}{get\+Settings()}} so other systems can read them without knowing the file format.
\item Auto-\/saves the project every {\ttfamily m\+\_\+auto\+Save\+Interval} seconds and debounces dirty writes via {\ttfamily m\+\_\+dirty\+Debounce}.
\end{DoxyItemize}
