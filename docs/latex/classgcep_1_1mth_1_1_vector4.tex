class gcep::\+mth::\+\+Vector4$<$ T $>$}
Represents a 4D mathematical vector with common arithmetic operations. 

Provides a complete set of vector operations such as addition, subtraction, normalization, dot product, interpolation, and reflection. Commonly used for homogeneous coordinates (XYZW) in 3D graphics.
