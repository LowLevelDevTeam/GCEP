\doxysubsection{Detailed Description}
Custom {\ttfamily std::\+streambuf} that tees all output into an Im\+Gui console log. 

Wraps an existing stream buffer (typically {\ttfamily std::\+cout\textquotesingle{}s}) and intercepts every character written to it. Characters are accumulated into a line buffer; on each newline the completed line is pushed into {\ttfamily m\+\_\+log} and the original buffer receives the character unchanged, preserving normal terminal output.

\begin{DoxyParagraph}{Usage}
Replace {\ttfamily std::\+cout.\+rdbuf()} with an instance of this class. Restore the original buffer on teardown via {\ttfamily Ui\+Manager::\+shutdown\+Console()}. 
\end{DoxyParagraph}
