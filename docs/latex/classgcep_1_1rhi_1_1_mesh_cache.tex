\doxysubsection{Detailed Description}
Maps canonical OBJ paths to their uploaded {\ttfamily \doxylink{structgcep_1_1rhi_1_1_geometry_region}{Geometry\+Region}} descriptors. 

Lives inside {\ttfamily Vulkan\+RHI} as a plain member. Ref-\/counted so GPU memory is only reclaimed when the last entity referencing a given OBJ is destroyed.
