\doxysubsection{Detailed Description}
Dear Im\+Gui panel that lets the user navigate the project\textquotesingle{}s content directory. 

{\ttfamily \doxylink{classgcep_1_1editor_1_1_content_browser}{Content\+Browser}} renders a two-\/column table:\+ the left column shows sub-\/directories and the right column shows files. Each entry is drawn as a centred icon with a truncated label underneath. The user can:\+
\begin{DoxyItemize}
\item Double-\/click a directory to descend into it.
\item Click the back button to ascend to the parent directory.
\item Drag a file entry onto a drop target (e.\+g. the material slot of an entity).
\item Receive a dropped path via {\ttfamily render\+Drop\+Target()}.
\end{DoxyItemize}

\begin{DoxyParagraph}{Root /\+ current path}
The browser starts at {\ttfamily default\+Path} (passed to the constructor) and never ascends above it. {\ttfamily m\+\_\+root\+Path} is fixed at construction; {\ttfamily m\+\_\+current\+Path} changes as the user navigates.
\end{DoxyParagraph}
\begin{DoxyParagraph}{Ownership}
{\ttfamily \doxylink{classgcep_1_1editor_1_1_content_browser}{Content\+Browser}} holds no heap resources beyond the {\ttfamily std::\+vector} caches that are refreshed on directory change. 
\end{DoxyParagraph}
