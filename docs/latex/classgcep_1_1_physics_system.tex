\doxysubsection{Detailed Description}
Singleton wrapper for the engine\textquotesingle{}s physics subsystem. 

\doxylink{classgcep_1_1_physics_system}{Physics\+System} manages the physics simulation, updating the physics world, syncing physics results to entity transforms, and performing queries like raycasts.

It internally owns a \doxylink{classgcep_1_1_physics_world}{Physics\+World} instance and interacts with the ECS Registry to manage Physics\+Components.
