\doxysubsection{Detailed Description}
Frustum plane data passed as a push constant to the culling compute shader. 

Populated each frame by {\ttfamily Vulkan\+RHI::\+extract\+Frustum()} using the Gribb-\/\+Hartmann method applied to the combined {\ttfamily proj} \texorpdfstring{$\ast$}{*} {\ttfamily view} matrix. Each plane is stored as a normalized {\ttfamily vec4} where {\ttfamily xyz} is the inward-\/facing normal and {\ttfamily w} is the signed distance from the origin, such that {\ttfamily dot(n, p)} + d \texorpdfstring{$>$}{>}= 0 means the point {\ttfamily p} is inside (or on) the plane.

\begin{DoxyNote}{Note}
The three {\ttfamily float} padding fields are required so the total struct size is a multiple of 16 bytes, satisfying the Vulkan push constant alignment rules. 
\end{DoxyNote}
