\doxysubsection{Detailed Description}
Owns a {\ttfamily Vk\+Swapchain\+KHR}, all swapchain {\ttfamily Vk\+Image} handles, and their {\ttfamily Vk\+Image\+Views}. 

\begin{DoxyParagraph}{Ownership model}
Swapchain images ({\ttfamily Vk\+Image}) are owned by the driver; this class stores them as plain {\ttfamily vk::\+\+Image} (non-\/owning). Image views ({\ttfamily Vk\+Image\+View}) are owned here via {\ttfamily vk::\+raii::\+\+Image\+View} and destroyed when this object is destroyed.
\end{DoxyParagraph}
\begin{DoxyParagraph}{Lifecycle}
Constructed by {\ttfamily \doxylink{classgcep_1_1rhi_1_1vulkan_1_1_vulkan_device}{Vulkan\+Device}} during {\ttfamily init\+Swapchain()}. On window resize, {\ttfamily \doxylink{classgcep_1_1rhi_1_1vulkan_1_1_vulkan_device_a95574ea7f13fe1a532e28494b8cc4a90}{Vulkan\+Device::\+recreate\+Swapchain()}} destroys the old instance (via {\ttfamily unique\+\_\+ptr::\+reset()}) before constructing a new one, preventing {\ttfamily Error\+Native\+Window\+In\+Use\+KHR}.
\end{DoxyParagraph}
\begin{DoxyParagraph}{Thread safety}
Not thread-\/safe. Used exclusively on the render thread. 
\end{DoxyParagraph}

Constructs the swapchain, creates all swapchain images, and creates image views. 

Selects the optimal surface format (prefers {\ttfamily B8\+G8\+R8\+A8\+\_\+\+SRGB} /\+ {\ttfamily e\+Srgb\+Nonlinear}), present mode (prefers Mailbox when {\ttfamily !desc}.vsync, falls back to FIFO), and extent (clamps {\ttfamily desc.\+width} /\+ {\ttfamily desc.\+height} to the surface capabilities). Requests a minimum of 3 images, clamped to the surface\textquotesingle{}s {\ttfamily max\+Image\+Count}.
