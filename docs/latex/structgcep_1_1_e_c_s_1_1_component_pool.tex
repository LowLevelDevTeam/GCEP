\subsubsection*{template$<$Component\+Concept T$>$\newline
class gcep::\+\+ECS::\+\+Component\+Pool$<$ T $>$}
Component pool implementation based on a Sparse Set data structure. 

Manages storage and lifecycle for components of type T. Uses three synchronized structures for optimal performance:\+
\begin{DoxyItemize}
\item A Paged Sparse Array for O(1) direct access via Entity\+ID.
\item A Dense Array for grouping active entities to allow fast iteration.
\item A contiguous data array to maximize CPU cache efficiency.
