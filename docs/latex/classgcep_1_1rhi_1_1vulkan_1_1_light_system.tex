\doxysubsection{Detailed Description}
Manages the GPU light SSBO, lighting compute pipeline, and the lit output image that replaces the raw scene color for TAA. 

\begin{DoxyParagraph}{Ownership}
{\ttfamily \doxylink{classgcep_1_1rhi_1_1vulkan_1_1_light_system}{Light\+System}} does not own the offscreen color /\+ depth images -\/ it receives {\ttfamily vk::\+\+Image\+View} handles from {\ttfamily \doxylink{classgcep_1_1rhi_1_1vulkan_1_1_vulkan_r_h_i}{Vulkan\+RHI}} and re-\/writes its descriptor set whenever those images are recreated (e.\+g. on resize).
\end{DoxyParagraph}

Records the lighting compute pass into {\ttfamily cmd}. 

Steps:\+
\begin{DoxyEnumerate}
\item Emits pre-\/dispatch barriers:\+ color + depth to {\ttfamily e\+Shader\+Read\+Only\+Optimal}, lit-\/output to {\ttfamily e\+General} (storage write).
\item Binds the lighting compute pipeline and descriptor set.
\item Pushes {\ttfamily \doxylink{structgcep_1_1rhi_1_1vulkan_1_1_lighting_push_constants}{Lighting\+Push\+Constants}} (inv\+View\+Proj, camera\+Pos, screen\+Size).
\item Dispatches (width+7)/\+8 \texorpdfstring{$\ast$}{*} (height+7)/\+8 thread groups.
\item Emits post-\/dispatch barrier:\+ lit-\/output to {\ttfamily e\+Shader\+Read\+Only\+Optimal} for subsequent TAA sampling.
\end{DoxyEnumerate}

Uploads point and spot light data to the GPU SSBO and meta UBO. 

Merges both lists into a single flat {\ttfamily \doxylink{structgcep_1_1rhi_1_1vulkan_1_1_g_p_u_light}{GPULight}} array (point lights first, then spot lights), clamps the total to {\ttfamily MAX\+\_\+\+LIGHTS}, and memcpys into the persistently-\/mapped device-\/visible buffers. Call once per frame before {\ttfamily \doxylink{classgcep_1_1rhi_1_1vulkan_1_1_light_system_a8fe0e9031d534e91ce3b371fec19a5d3}{record\+Lighting\+Pass()}}.
