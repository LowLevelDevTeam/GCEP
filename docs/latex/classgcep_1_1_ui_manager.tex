\doxysubsection{Detailed Description}
Top-\/level editor UI manager built on Dear Im\+Gui and Im\+Guizmo. 

{\ttfamily \doxylink{classgcep_1_1_ui_manager}{Ui\+Manager}} owns the full Im\+Gui per-\/frame lifecycle and renders the editor shell:\+ a docked layout containing a 3D viewport, scene hierarchy, entity properties, scene settings, audio controls, content browser, and console.

\begin{DoxyParagraph}{Owned subsystems}

\begin{DoxyItemize}
\item {\bfseries{Im\+Gui /\+ Im\+Guizmo}} — context, GLFW + Vulkan backends, dockspace.
\item {\bfseries{Console redirection}} — {\ttfamily \doxylink{classgcep_1_1_im_gui_console_buffer}{Im\+Gui\+Console\+Buffer}} tees {\ttfamily std::\+cout} /\+ {\ttfamily std::\+cerr} into {\ttfamily m\+\_\+console\+Items}.
\item {\bfseries{Physics}} — holds a reference to the singleton {\ttfamily \doxylink{classgcep_1_1_physics_system}{Physics\+System}}; starts /\+ pauses /\+ stops simulation from the viewport toolbar.
\item {\bfseries{Audio}} — holds a pointer to {\ttfamily \doxylink{classgcep_1_1_audio_system}{Audio\+System}} and a list of loaded {\ttfamily \doxylink{classgcep_1_1_audio_source}{Audio\+Source}} handles managed from the audio control panel.
\end{DoxyItemize}
\end{DoxyParagraph}

Initialises the editor UI and binds it to the application systems. 

Steps performed:\+
\begin{DoxyEnumerate}
\item Stores {\ttfamily window}; obtains the {\ttfamily \doxylink{classgcep_1_1_audio_system}{Audio\+System}} singleton.
\item Creates the Im\+Gui context with docking and multi-\/viewport flags enabled.
\item Loads {\ttfamily Nunito-\/\+Regular.\+ttf} at 18 pt and merges Font Awesome icons.
\item Calls {\ttfamily Im\+Gui\+\_\+\+Impl\+Glfw\+\_\+\+Init\+For\+Vulkan()} and {\ttfamily Im\+Gui\+\_\+\+Impl\+Vulkan\+\_\+\+Init()}.
\item Calls {\ttfamily init\+Console()} to redirect {\ttfamily std::\+cout} /\+ {\ttfamily std::\+cerr}.
\end{DoxyEnumerate}
