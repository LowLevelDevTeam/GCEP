Detailed Description

Dear ImGui panel that lets the user navigate the project's content directory.

ContentBrowser renders a two-column table: the left column shows sub-directories and the right column shows files. Each entry is drawn as a centred icon with a truncated label underneath. The user can:

  • Double-click a directory to descend into it.
  • Click the back button to ascend to the parent directory.
  • Drag a file entry onto a drop target (e.g. the material slot of an entity).
  • Receive a dropped path via renderDropTarget().
Root / current path
The browser starts at defaultPath (passed to the constructor) and never ascends above it. m_rootPath is fixed at construction; m_currentPath changes as the user navigates.
Ownership
ContentBrowser holds no heap resources beyond the std::vector caches that are refreshed on directory change.
