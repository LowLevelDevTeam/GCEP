Detailed Description

Hash specialisation for gcep::rhi::vulkan::Vertex enabling its use as an unordered_map key during OBJ vertex deduplication.

Combines the GLM hashes of pos, normal, and texCoord using the standard hash-combine XOR/shift pattern. Requires GLM_ENABLE_EXPERIMENTAL and <glm/gtx/hash.hpp>.
