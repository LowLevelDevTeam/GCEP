Detailed Description

High-level wrapper around a Jolt Physics shape.

PhysicsShape acts as an abstraction layer over Jolt's internal JPH::Shape representation. It allows the engine to remain decoupled from the physics backend while exposing a clean and controlled API.

Instances of this class are typically shared across multiple physics bodies through std::shared_ptr, and are commonly cached to avoid recreating identical shapes.
