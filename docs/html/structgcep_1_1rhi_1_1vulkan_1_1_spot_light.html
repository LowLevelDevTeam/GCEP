Detailed Description

CPU-side description of a spot light.

A spot light illuminates a cone defined by innerCutoffDeg and outerCutoffDeg (both in degrees, measured from the central axis). Between the two angles the intensity is smoothly attenuated using a cosine-based falloff. Beyond radius the light has no influence.
