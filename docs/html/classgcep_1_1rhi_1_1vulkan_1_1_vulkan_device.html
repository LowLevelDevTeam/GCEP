Detailed Description

Central Vulkan device manager.

Owns the Vulkan instance, debug messenger, surface, physical device, logical device, queue, swapchain, and all per-frame-in-flight contexts. Coordinates the full frame lifecycle (acquire -> record -> submit -> present).

Init sequence
createVulkanDevice() must be the first call after construction. It runs all eight init stages in mandatory order: initInstance -> initDebugMessenger -> initSurface -> selectPhysicalDevice -> initLogicalDevice -> initSwapchain -> initFrameContexts.
Frame lifecycle
if (!device.beginFrame()) return; // fence wait + acquire
// ... record via currentCommandBuffer() ...
device.endFrame(); // submit + present
Thread safety
All public methods must be called from a single render thread.

◆ beginFrame()

bool gcep::rhi::vulkan::VulkanDevice::beginFrame ( )
nodiscard

Begins a new frame: waits on the in-flight fence, acquires a swapchain image, and opens the frame's command buffer for recording.

Selects the next frame slot (round-robin over MAX_FRAMES_IN_FLIGHT), calls waitAndReset() on its VulkanFrameContext to stall the CPU until the GPU is done with that slot, then signals presentCompleteSemaphore via vkAcquireNextImageKHR and calls beginRecording() on the command list.

Returns
true if recording can proceed; false if the swapchain is out-of-date (eErrorOutOfDateKHR), in which case the caller must skip the rest of the frame.
Exceptions
std::runtime_erroron any acquire result other than eSuccess, eSuboptimalKHR, or eErrorOutOfDateKHR.

◆ endFrame()

void gcep::rhi::vulkan::VulkanDevice::endFrame ( )

Ends the current frame: closes the command buffer, submits it, and presents.

Calls endRecording() on the command list, submits to the graphics queue with presentCompleteSemaphore as the wait semaphore and m_renderFinishedSemaphores[m_imageIndex] as the signal semaphore, then calls vkQueuePresentKHR. Advances m_frameIndex.

Exceptions
std::runtime_errorif vkQueuePresentKHR returns a result other than eSuccess or eSuboptimalKHR.

Destroys and recreates the swapchain for a new framebuffer size.

Calls waitIdle(), destroys the existing VulkanSwapchain (via unique_ptr::reset() to avoid ErrorNativeWindowInUseKHR), then constructs a new one at width × height. Also rebuilds the per-image acquire semaphores to match the new image count.

Parameters
widthNew framebuffer width in pixels.
heightNew framebuffer height in pixels.
