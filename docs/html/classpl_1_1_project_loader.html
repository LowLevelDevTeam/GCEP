Detailed Description

Singleton responsible for project lifecycle management.

ProjectLoader is the single point of truth for the active project. It:

  • Presents an ImGui project-selection / creation UI at startup.
  • Reads and writes the .gcepproj JSON file via RapidJSON.
  • Exposes runtime settings through getSettings() so other systems can read them without knowing the file format.
  • Auto-saves the project every m_autoSaveInterval seconds and debounces dirty writes via m_dirtyDebounce.

◆ update()

void pl::ProjectLoader::update ( double deltaTime)

Advances auto-save and dirty-debounce timers.

Should be called once per frame. When m_dirty is true and at least m_dirtyDebounce seconds have elapsed since the last modification, calls saveProject(). Also triggers saveProject() every m_autoSaveInterval seconds regardless of the dirty flag.

Parameters
deltaTimeElapsed time since the previous frame, in seconds.
