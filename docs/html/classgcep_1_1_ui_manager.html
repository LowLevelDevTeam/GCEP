Detailed Description

Top-level editor UI manager built on Dear ImGui and ImGuizmo.

UiManager owns the full ImGui per-frame lifecycle and renders the editor shell: a docked layout containing a 3D viewport, scene hierarchy, entity properties, scene settings, audio controls, content browser, and console.

Owned subsystems
  • ImGui / ImGuizmo — context, GLFW + Vulkan backends, dockspace.
  • Console redirection — ImGuiConsoleBuffer tees std::cout / std::cerr into m_consoleItems.
  • Physics — holds a reference to the singleton PhysicsSystem; starts / pauses / stops simulation from the viewport toolbar.
  • Audio — holds a pointer to AudioSystem and a list of loaded AudioSource handles managed from the audio control panel.
Frame sequence (uiUpdate)
beginFrame()
if (!m_dockspaceInitialized)
{
setupViewport() // fullscreen host window
getDockspaceID() // dockspace + main menu bar
initDockspace() // one-time layout split (first frame only)
}
drawViewport() // offscreen texture + gizmo overlay + toolbar
drawSettings() // render settings, camera, lighting, grid
drawSceneHierarchy() // entity list + spawn / remove buttons
drawEntityProperties() // transform / texture / physics / gizmo controls
drawAudioControl() // per-source playback controls
drawConsole() // scrolling log + text input
// push scene and camera data to the RHI

Note
All methods must be called from the main / render thread.

Initialises the editor UI and binds it to the application systems.

Steps performed:

  1. Stores window; obtains the AudioSystem singleton.
  2. Creates the ImGui context with docking and multi-viewport flags enabled.
  3. Loads Nunito-Regular.ttf at 18 pt and merges Font Awesome icons.
  4. Calls ImGui_ImplGlfw_InitForVulkan() and ImGui_ImplVulkan_Init().
  5. Calls initConsole() to redirect std::cout / std::cerr.
Parameters
windowThe application GLFW window. Must outlive this object.
managerPointer to the active scene manager. Must outlive this object.
reloadReference to the application reload flag; set to true when the user requests a project reload.
closeReference to the application close flag; set to true when the user requests exit.

◆ drawGizmo()

void gcep::UiManager::drawGizmo ( )

Renders the ImGuizmo manipulation widget over the viewport.

Early-returns if no entity is selected. Reads the selected entity's model matrix, calls ImGuizmo::Manipulate(), decomposes the result, and writes position / rotation / scale back into the ECS ECS::Transform component.

◆ drawGizmoControls()

void gcep::UiManager::drawGizmoControls ( )

Renders gizmo operation / mode / snapping controls in the properties panel.

Shows a combo for Translate / Rotate / Scale, a snap checkbox, and per-operation snap value inputs. Called by drawEntityProperties() when the simulation is not running.

◆ handleGizmoInput()

void gcep::UiManager::handleGizmoInput ( )

Handles keyboard shortcuts that change the active gizmo operation.

W = Translate, E = Rotate, R = Scale. Ignored when any ImGui window is unfocused, when ImGuizmo::IsUsing() is true, or while the simulation is actively playing.
