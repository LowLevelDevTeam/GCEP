Enumerates all supported physics collision shape types.

This enum describes the high-level abstraction of collision shapes used by the engine. Each type maps internally to a specific Jolt Physics shape implementation.

◆ setDarkTheme()

void gcep::setDarkTheme ( )
inline

Applies the GCEP dark colour theme to the active ImGui context.

Sets geometry, spacing, border, rounding, and every ImGuiCol_* entry to produce a consistent dark editor look. Safe to call at any time between ImGui::NewFrame() calls; changes take effect on the next rendered frame.
