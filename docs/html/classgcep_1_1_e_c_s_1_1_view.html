Detailed Description

template<typename... Args>
class gcep::ECS::View< Args >

Query system used to iterate over entities possessing a specific set of components.

The View is the primary tool for entity filtering. It optimizes iteration by identifying the smallest (most restrictive) pool to minimize the number of checks performed.

Template Parameters
ArgsList of component types required for filtering.
