◆ spawnAsset()

ECS::EntityID gcep::editor::spawnAsset ( SLS::Scene & scene,
rhi::vulkan::VulkanRHI * rhi,
const std::string & path,
glm::vec3 pos )

Spawns an external OBJ asset into scene at position pos.

Loads the mesh from path through the RHI, creates an ECS entity with a MeshComponent and a Transform initialised to pos, and registers the entity with the scene manager.

Parameters
sceneTarget scene that will own the new entity.
rhiPointer to the active Vulkan RHI used to upload the mesh data.
pathFilesystem path to the OBJ (or supported mesh) file.
posWorld-space position at which the asset is spawned.
Returns
The ECS entity ID of the newly created entity.
