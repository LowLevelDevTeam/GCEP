Detailed Description

Singleton wrapper for the engine's physics subsystem.

PhysicsSystem manages the physics simulation, updating the physics world, syncing physics results to entity transforms, and performing queries like raycasts.

It internally owns a PhysicsWorld instance and interacts with the ECS Registry to manage PhysicsComponents.
