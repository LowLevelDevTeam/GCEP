Detailed Description

Per-mesh data stored in the GPU-side SSBO (set 0, binding 0).

One entry per mesh, indexed in the vertex shader via SV_StartInstanceLocation which carries the firstInstance value set in each DrawIndexedIndirectCommand.

Must be kept byte-for-byte identical to the ObjectData / GPUMeshData struct in both Base_Shader.slang and Cull.slang.

Note
The two float padding fields (_pad0, _pad1) are required to satisfy the std430 alignment rule for vec3 members inside a storage buffer.
